# DDIC Contrast Reports

This directory stores generated validation artifacts for the DDIC contrast enhancement workflow.

## Intended contents
- branch comparison reports
- validation reports
- pitfall checklists
- handoff summaries for DDIC implementation teams

## Reporting rules
- keep generated reports versioned by date or round number
- keep metric names consistent with `scheme3/src/ce_scheme3/metrics.py`
- include both still-image and temporal-sequence evidence when available
