# DDIC Contrast Pitfall Checklist

## Common failure modes
- highlight whitening
- shadow noise boost
- false contour amplification
- gray-black lifting
- over-enhancement on already high-contrast scenes
- frame flicker on slow luminance drift
- scene-cut instability
- skin-tone unnaturalness
- subtitle or UI halo emphasis

## Review rule
For each report round, map every failure mode to at least one synthetic pattern and one real-image or temporal-sequence sample when available.
